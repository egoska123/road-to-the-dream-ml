# Road to the ML

Этот репозиторий — моя практическая дорожная карта от Frontend dev к Junior ML / AI Engineer.

## Что внутри

- mini-проекты по classic ML
- mini-проекты по PyTorch / deep learning
- NLP и Transformers
- RAG и retrieval
- recommender systems
- инженерная обвязка: FastAPI, Docker, GitHub, reproducibility

## Структура

- `projects/` — все проекты
- `docs/notes/` — заметки по ходу обучения
- `docs/templates/` — шаблоны для новых проектов
- `docs/interview/` — шпаргалки и разборы
- `shared/` — переиспользуемые функции
- `archive/` — старые, замороженные или черновые вещи
